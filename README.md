# 🍅 Tomatoes Quality Detection

A machine learning + IoT project to detect fresh vs rotten tomatoes using **Raspberry Pi**and **Streamlit** for visualization.

## 🚀 Tech Stack
- **Python 3.10+**
- **Flask / Streamlit**
- **OpenCV** (for image processing)
- **Raspberry Pi** Run the model
